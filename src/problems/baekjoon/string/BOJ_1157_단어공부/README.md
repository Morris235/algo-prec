## 접근 방식
- **HashMap 풀이:** `getOrDefault()`로 문자별 빈도를 카운트, `Collections.max()`로 최빈값 찾음.  
- **Array 풀이:** `int[26]` 배열을 이용해 아스키 코드 기반으로 카운트, 한 번의 순회에서 최댓값과 동률 처리.  

---

## 복수 풀이 비교
- HashMap: 직관적이고 다른 문자/유니코드에도 응용 가능.  
- Array: 이 문제처럼 알파벳에 한정되면 성능적으로 훨씬 유리.
