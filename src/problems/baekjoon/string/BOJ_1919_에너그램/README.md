입력: aabbcc, xxyybb
출력: 8

## 접근 방식

- 입력 문자열 s1, s2를 각각 순회하며 int[26] cnt에 카운트(+1/-1)
  - s1의 각 문자: cnt[c - 'a']++
  - s2의 각 문자: cnt[c - 'a']--
  - 절대값으로 합산하여 삭제해야할 갯수를 구함
